# 손금 수명 미리보기 Layout Structure

## Screen Flow

`home -> capture -> result`

## Home

```text
┌──────────────────────────────────────┐
│ Header: 손금 수명 미리보기 + subtitle     │
├──────────────────────────────────────┤
│ Hero card                            │
│ - 앱 설명                             │
│ - 오락용 안내                         │
├──────────────────────────────────────┤
│ 진행 방식 카드 (3 step)              │
├──────────────────────────────────────┤
│ 권한 상태 카드                        │
│ - 현재 권한 문구                      │
│ - 오류/안내 메시지                    │
├──────────────────────────────────────┤
│ CTA                                  │
│ - 손금 찍기 시작                      │
│ - 카메라 권한 다시 요청 (조건부)      │
└──────────────────────────────────────┘
```

## Capture

```text
┌──────────────────────────────────────┐
│ Back (홈으로)                         │
├──────────────────────────────────────┤
│ Preview Frame                         │
│ - 촬영 전: 가이드 아이콘 + 문구        │
│ - 촬영 후: 손금 이미지                 │
├──────────────────────────────────────┤
│ 권한 경고 카드 (권한 거부 시)         │
│ - 권한 허용하기 버튼                   │
├──────────────────────────────────────┤
│ 상태 메시지                           │
├──────────────────────────────────────┤
│ CTA                                  │
│ - 손금 촬영 / 다시 촬영                │
│ - 결과 분석하기 (촬영 후)              │
│ - 촬영 이미지 지우기 (촬영 후)         │
└──────────────────────────────────────┘
```

## Result

```text
┌──────────────────────────────────────┐
│ Tier card                             │
│ - 티어 라벨 + 헤드라인 + 요약         │
├──────────────────────────────────────┤
│ Capture snapshot + generatedAt        │
│ - 오락용 디스클레이머                 │
├──────────────────────────────────────┤
│ Score panel                           │
│ - 생기/안정/회복/집중 progress bar     │
├──────────────────────────────────────┤
│ 핵심 해석 리스트                       │
├──────────────────────────────────────┤
│ 추천 액션 리스트                       │
├──────────────────────────────────────┤
│ CTA                                  │
│ - 결과 공유하기                        │
│ - 사진 다시 찍고 재분석                │
│ - 처음부터 다시                        │
└──────────────────────────────────────┘
```

## Differentiation Notes

- 권한 요청을 홈/촬영 양쪽에서 분리해 실패 경로를 명확히 제공
- 촬영 결과 기반 deterministic 로컬 점수 계산
- 공유는 deep link 기반 텍스트 공유로 확장
